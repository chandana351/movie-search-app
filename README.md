# Movie Search App

A responsive React movie search app with search, genre filtering, rating sorting and movie cards.

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL shown by Vite.

## Build

```bash
npm run build
```

## GitHub Pages

This project is configured with:

```js
base: "/movie-search-app/"
```

Deploy from the `dist` folder or use a GitHub Pages workflow.
